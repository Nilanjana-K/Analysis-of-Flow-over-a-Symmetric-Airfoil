**Kutta-Zhukovsky Airfoil Flow Visualization**

**Overview**

This MATLAB project demonstrates the transformation of potential flow over a circular cylinder into flow over a symmetric airfoil using the Joukowski conformal mapping technique.

**Features**

* Potential flow over a cylinder
* Joukowski airfoil generation
* Streamline visualization
* Stagnation point plotting
* Subplot comparison of circle and airfoil flow

**Concepts Used**

* Potential Flow Theory
* Stream Function
* Complex Variables
* Conformal Mapping
* Joukowski Transformation

**Governing Transformation:**
Kutta - Zhukovsky Transformation

**Notes**

* Inviscid and incompressible flow assumption
* No viscous drag or boundary layer effects included
